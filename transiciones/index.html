Proceso de renderizado:

Capas

js/ css – layout – paint – composite


  • layout: es el mśs costoso. Se encarga de establecer todos las propiedades del modelo de caja. Si se modifica una propiedad del layout se modificara toda la geometría de la página.

  • Paint: es la capa que aplica la “pintura” de las etiquetas de html, colores, imagenes, etc.

  • composite: se encarga de darle el orden a las capas del navegador. Hace referencia a los elementos que no necesitan layout o paint. Uno de las propiedades que ocupa es transform ya que da el orden a las capas del navegador y comprime y dibuja en la pantalla

cuando se modifican las propiedades del layout se vuelve a renderizar todo la página
